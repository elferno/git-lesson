git clone https://... - клонировать к себе репозиторий git status - статус текушего локального репозитория git pull - забрать свежие обновления git init - создать новый пустой репозиторий git add path-to-file || folder/ - добавить новый локальный файл / фолдер в этот репозиторий git remote add origin https://... - указываем путь к репозиторию гита, куда будут закачаны наши файлы git commit -am "version 0.0.1" - зафиксиравть изменения локального репозитория и дать им комментарий git push -u origin master - отправить все закомиченные изменения в репозиторий на сервер
