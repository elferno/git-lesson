git clone path-to-repository - клонировать к себе репозиторий git status - статус текушего локального репозитория git add path-to-file - добавить новый локальный файл под контроль GIT'a git commit -am "commentary-text" - зафиксиравть изменения локального репозитория и дать им комментарий git push - отправить все закомиченные изменения в репозиторий на сервер git pull - забрать свежие обновления
